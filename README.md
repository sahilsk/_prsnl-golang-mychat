About
--------

A little very basic server-client chat application using golang core packages.

How-to
------

- Build the binary first

       go build -o mychat

       - or run it directly

            go run main.go

### BUT don't forget the arguments

    -listen
            listen to the specified ip address

## Start Server


    ./mychat -listen 127.0.0.1


### Start client

    ./mychat 127.0.0.1



### Lastly, Have fun
